Jet9 manage shell
=================

Requirements:
-------------

1. python peewee orm:
		
```
# sudo pip install peewee
```


	
2. jet9 modules installed or placed in work directory:

```
jet9api
jet9cmd
```



Installation:
-------------
	
    # create db
    jet9 db create
	
    # add tariffs
    jet9 tariff ...

    # add users
    jet9 user ...
    
    ...and so on
